### HTML CSS 굵기 지정
HTML 태그로 굵게 지정을 했어도 CSS를 둘 다 신경 써야함

### 글자 크기
크기를 표현하는 단위가 다양함

### font-size
  - 절대값 : px(픽셀), pt(포인트)
  - 상대값 : %, em, rem
    - 부모의 크기에 비례함

### pt(포인트)
pt는 웹사이트를 종이로 인쇄할 경우 결과물의 크기를 결정하는 단위임
인쇄할 때 아니면 안 씀

### text-decoration
  - 밑줄, 취소선 윗줄, 물결선, 점선, ----

### text-transform
  - 첫 알파벳 대문자, 전체 대문자, 전체 소문자
  - DB 때문에 어떤 문자형태로 올지 모르기 때문에 보험용으로 작성

### CSS로 문단 스타일
  - CSS로 문단과 목록에 스타일을 줄 수 있음
  - 종류로는 left right center justify가 있음
  - 자주 쓰는 건 justify는 양쪽 정렬임 

### letter-spacing / word-spacing / line-height / text-indent
  - letter-spacing : 글자간
  - word-spacing : 띄어쓰기
  - line-height : 줄 높이
  - text-indent : 첫 p태그 띄어쓰기
