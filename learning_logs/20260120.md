## 요소의 크기
  - px로 하면 반응형이 아니라서 불편함
  - %로 하면 부모 크기에 비례함
  - viewport를 이용하면 유동적으로 크기를 조절 가능
    - vw : viewport width / 1~100까지
    - vh : viewport height / 1~100까지
    - vmax : viewpoert maximum / 상대적으로 길이가 긴 쪽
    - vmin : viewport minimum / 상대적으로 짧은 쪽
    - vmax / vmin은 어떤 디바이스인지에 다름 컴퓨터는 가로 길이지만 폰은 세로 길이임
    - vw / vh는 텍스트 크기를 설정할 수 있음 
  - calc(100% + 100px) 100%의 100px 더 큰 값을 낼 수 있음
