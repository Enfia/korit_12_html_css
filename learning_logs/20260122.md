# 용도에 따른 태그 사용하기
  - `<!DOCTYPE html>` : html5 형식을 명시
  - `<head>` : 눈으로 볼 수 없지만 컴퓨터가 읽을 수 있는 문서의 정보와 사양이 담겨있음
    - `<title>` : 사이트 제목임
    - `<meta>`
      1. charset : 어떤 방식으로 읽을 지를 명시
      2. viewport : 화면에 보여지는 영역
      3. Open Graph : 웹사이트 제목과 썸네일 등 기본 정보를 태그로 정의
    - `favicon` : 웹사이트 제목 옆에 이미지 넣기

# 시멘틱 태그의 장점
  - HTML5에서 활용하여 웹사이트를 구성
  - 역할에 따라서 이름이 다르게 붙여진 div임
  - 웹 접근성 개선
  - SEO 장점 : 검색엔진이 노출시켜줌
  - 유지보수와 가독성에 유리함

## 시멘틱 태그의 종료
  1. `<header>`
    - 필요하다면 사용 가능
  2. `<nav>`
    - 링크로 이동하는 요소를 포함하고 있음
  3. `<footer>`
    - 최하단에 보여지는 요소
  4. `<main>`
    - body 태그 안에 있음 페이지의 가장 중요한 부분
  5. `<aside>`
    - 주로 사이드바 역할을 함
  6. `<section>`
    - 컨텐츠나 주제, 작은 단위로 나눌 때 `<div>`을 사용함
  7. `<article>`
    - 주로 기사, 블로그 포스트, 댓글임
  8. `<figure>`
    - `<img>` 태그와 `<figcaption>` 태그를 묶을 때 사용함

## CSS 변수 사용하기
  - 원하는 색에 변수명을 사용할 수 있음
  - 대쉬 2개로 변수를 표시 가능

  - 작은 텍스트 크기 `--font-small`
