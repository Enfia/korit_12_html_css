### box-sizing
  - 속성은 content-box(padding 제외) / border-box(padding 포함) 하나로 고정 시켜야함

### border-radius
  - 값 하나만 : 네 곳 모서리에 적용됨
  - 값 두 개만 : 상단 왼쪽과 하단 오른쪽 -> 상단 오른쪽 하단 왼쪽
  - 값 네 개만 : 왼쪽 상단 모서리부터 시계 방향

### overflow
  - 기본 값은 auto로 되어있음
  - auto : 자식 요소가 부모의 요소보다 큰 경우 스크롤 바가 생김
  - visible : 부모 영역을 벗어난 자식 요소를 그대로 보여주도록 허용
  - hidden : 부모 영역을 벗어난 부분은 보이지 않도록 함
  - scroll : auto와 비슷함 차이점은 부모 영역을 넘어가지 않아도 스크롤바 보여줌
  - 속성값 두 개 입력 가능 첫 번째는 가로 두 번째는 세로
    - overflow-x : 가로
    - overflow-y : 세로

### div를 span처럼 쓰기
  - display쓰면 span 태그처럼 쓸 수 있음

### position static(기본값)
  - 페이지의흐름을 따라감
  - top / botom / left / right / z-index의 위치 속성의 영향을 받지 않음

### position relative
  - 원래 있던 위치 기준으로 속성값이 적용됨
  - top / bottom / left / right / z-index의 위치 속성의 영향을 받음
  - 공백의 위치는 그대로 유지됨

### position absolute
  - static이 아닌 바로 위 부모 요소 기준으로 top / bottom / left / right / z-index의 위치 속성의 영향을 받음

### position fixed
  - 부모 요소가 아니라 viewport를 기준으로 위치를 지정함
  - navbar에 사용함

### position sticky
  - 고정되어 있다가 fixed 됨
  - 부모의 영역을 넘어가지 못함

### z-index
  - 겹치게 할 수 있음 값이 높을 수록 위임
  - 기본값은 auto임(값은 0)

### flex 레이아웃
