### iframe 태그 문제점 
1. 보안 문제 : 아이디와 비밀번호를 해킹 당할 수 있음 이를 clickjacking이라고 함
2. 사용성 저하 : 뒤로 가기 등이 적용되지 않음 
3. SEO에 최적화되지 않음 : SEO와 관계가 없음 
4. 브라우저 새창 문제 : 여러 사이트를 불러오는 것과 같아서 RAM을 많이 잡아먹음

### `<div>` vs `<span>` 아무것도 하지 않는 태그들
